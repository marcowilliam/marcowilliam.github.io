Human Tetris

Kinect Project done using Processing 2.0.

This was the final project done for the Physical Computing class. We needed to mix Kinect and Processing 2.0. So I decided to make a fun game.

The image above is the one used to promote the game during an exposition in the end of the spring quarter, 2015 at SCAD.

Basically the game is kind of a Tetris but with body positions. Summing up, some bodies positions are going to fall from the top of the screen and the player has to copy them before they touch the floor. The more positions are caught the more chances of beating the highest score exist.

Bellow there is a short video that shows a bit how the game works.

Its code can be found on my github.

  • Professor: Josephine Leong
  • Course: Physical Computing

Can You Memorize It?

Arduino project

This project was made during the Physical Computing classes. We needed to make anything with the knowlegdes learned about Arduino. So I made a game based on the game “Donkey Kong Country 3: Dixie Kong's Double Trouble!”, where there was a part of the game that the player has to memorize the sequence that the lights were blinked and than repeat it.

The game has basically 3 functionalities:

  • When the player is able to play, the RGB LED will keep being green.
  • If the player loses, it will show what would be the right LED to be selected, the RGB LED will be red and a game over song is played.
  • When the player reaches 20 points he/she wins the game and a celebration happens

Bellow there are two videos showing how the game works. The first one is the game itself



The second one is when the palyer reaches 20 points and win the game


Its code can be found on my github.

  • Professor: Josephine Leong
  • Course: Physical Computing

Game done using Processing 2.0
